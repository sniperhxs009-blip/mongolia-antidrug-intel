# 全量修改汇总清单（定稿综合整改）

> 项目：mongolia-antidrug-intel  
> 原则：不删减七大体系 / 定时巡检 / 邮件简报 / 多语种翻译 / PDF 统计；仅降噪、合规、修复、加固。  
> **部署策略：人工确认后推送，禁止自动上线 Render。**

---

## 一、检索降噪

| 文件 | 修改位置/内容 | 修复目标 |
|------|---------------|----------|
| `config/core_official.py` | `SEARCH_NEGATIVE_EXCLUDE`；删除 `site:gov.mn` / 快照包装；`is_forbidden_url` 含 gov.mn 即拦 | 剔除内蒙古/俄边境/烟草/兴奋剂噪音；合规禁访 gov.mn |
| `config/drug_lexicon.py` | `_finalize_query` 追加负面排除 + 强制蒙古国/乌兰巴托/扎门乌德/甘其毛都绑定 | 检索语句地域强制绑定 |
| `config/global_media.py` | 导入并追加 `SEARCH_NEGATIVE_EXCLUDE` | 全球媒体检索降噪 |
| `config/forum_search.py` | 同上；论坛默认不启用由配置控制 | 论坛噪音可控 |
| `config/sources.py` | SEARCH_FEEDS 默认 when=`30d` | 时效窗口收紧 |
| `config/official_stats.py` | 清除 gov.mn 统计检索 | 合规 |
| `.env` / `.env.example` | `NEWS_WHEN`/`FULL_WHEN`/`FORUM_WHEN`=`30d`；`ENABLE_FORUM_SEARCH=false`；`CRAWL_LOOSE_FILTER=false` | 默认降噪 |
| `app/crawler/search_feeds.py` | 标题强毒品词前置过滤；论坛默认关；入库前严格蒙古判定；填充可信度/告警/口岸/毒品种类 | 减少无效下载与脏入库 |
| `app/crawler/filters.py` | 强/弱词分层；`title_has_strong_drug`；扩充 `HARD_EXCLUDE` | 仅弱词不入库 |
