# 多引擎搜索与论坛社区采集

## 当前已接入的搜索通道

| 引擎 | 用途 | 时效 |
|------|------|------|
| Google News RSS | 主新闻聚合（中/英/蒙/俄） | 默认 `when:1y`（7d/30d 实测几乎无蒙古涉毒新闻） |
| Bing News RSS | 补充新闻 | 随 `when` 映射（默认近一年） |
| DuckDuckGo 新闻/网页 | 补充搜索 | 随 `when` 映射 |
| Reddit JSON API | 社区帖子 | `t=year`（默认）+ 发布时间过滤 |
| Google/DDG 网页搜索 | Reddit/Quora/知乎/贴吧等论坛链 | `when` + `tbs=qdr:y` |
| 蒙古媒体站内搜 | GoGo / Montsame / IKON / News.mn | 站内结果再过滤 |

## 时效控制（平衡稀疏信源）

蒙古国公开涉毒新闻极少：`when:7d` / `30d` 实测 Google News 可入库条数≈0。  
默认改为近一年，并用内容过滤器（蒙古+涉毒双条件）挡旧噪声，而不是靠过短时间窗把库清空。

1. Google 类查询带 `when:1y`（可用环境变量改回 `7d`/`30d`/`90d`）  
2. `CRAWL_MAX_AGE_DAYS=365`：有发布时间且超过阈值才丢弃  
3. Reddit / Bing / DDG 时间参数与 `when` 对齐  

## 论坛覆盖

Reddit、Quora、Bluelight、Drugs-Forum、Medium、知乎、百度贴吧等（公开可检索帖）。

## 开关

```env
ENABLE_FORUM_SEARCH=true
NEWS_WHEN=1y
FULL_WHEN=1y
CRAWL_MAX_AGE_DAYS=365
```
