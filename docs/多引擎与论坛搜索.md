# 多引擎搜索与论坛社区采集

## 当前已接入的搜索通道

| 引擎 | 用途 | 时效 |
|------|------|------|
| Google News RSS | 主新闻聚合（中/英/蒙/俄） | `when:7d` / `when:30d` |
| Bing News RSS | 补充新闻 | 近 7 天 |
| DuckDuckGo 新闻/网页 | 补充搜索 | 近月 |
| Reddit JSON API | 社区帖子 | `t=month` + 发布时间过滤 |
| Google/DDG 网页搜索 | Reddit/Quora/知乎/贴吧等论坛链 | `when` + `tbs=qdr:m` |
| 蒙古媒体站内搜 | GoGo / Montsame / IKON / News.mn | 站内结果再过滤 |

## 严格近期控制

1. 所有 Google 类查询强制带 `when:7d`（新闻）或 `when:30d`（全量）  
2. `CRAWL_MAX_AGE_DAYS=30`：有发布时间的条目超过 30 天直接丢弃  
3. Reddit 使用 `sort=new&t=month`  
4. Bing 使用 `interval=7`  

避免再出现 2012 恐龙化石、2016 奢侈品走私等旧闻。

## 论坛覆盖

Reddit、Quora、Bluelight、Drugs-Forum、Medium、知乎、百度贴吧等（公开可检索帖）。

## 开关

```env
ENABLE_FORUM_SEARCH=true
NEWS_WHEN=7d
FULL_WHEN=30d
CRAWL_MAX_AGE_DAYS=30
```
