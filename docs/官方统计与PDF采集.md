# 官方统计与 PDF 报表采集

系统在新闻监测之外，增加**官方统计与年报体系（system_id=9）**。

## 覆盖机构

| 机构 | 作用 |
|------|------|
| 总检察院 prosecutor.mn | 涉毒案件数、起诉/判决、同比 |
| 海关总局 customs.gov.mn | 口岸查获、走私批次 |
| 警察总局 police.gov.mn | 立案与专项行动 |
| 国家统计局 1212.mn | 犯罪相关公开统计 |
| 法律信息中心 legalinfo.mn | 管制法规与说明材料 |
| UNODC | 国别/世界毒品报告 PDF |
| 国家广播 MNB | 转述检察院官方数字的权威通稿 |

## 两种采集

1. **统计新闻搜索**（每小时新闻监测也会跑）  
   专门搜「检察院/海关 + 涉毒 + 统计/同比」类稿件，并从正文抽取数字入库。

2. **PDF 报表**（每日全量巡检）  
   `filetype:pdf` 定向搜索官方域名，下载后用 `pypdf` 抽文本，再抽取统计点。

## 结构化字段（`stat_records`）

- 涉毒案件数、同比增长率、涉案人数、判决人数  
- 查获次数、隐匿滥用人数估算等  
- 带时期（如 `2025`、`2026-Q1`）、原文片段、来源 URL

## API

- `GET /api/stat-records` — 最新统计点列表  
- `GET /api/stats` — 含 `stat_records` 计数  

指挥台「官方统计数字」面板每 30 秒自动刷新。

## 开关

```env
ENABLE_OFFICIAL_STATS=true
```
