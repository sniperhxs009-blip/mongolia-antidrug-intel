## 产量优化说明（2026-07）

在既有合规约束不变的前提下，本轮对检索、过滤、抓取与研判做了增量优化，目标是显著提升有效情报条数。

### 硬性约束（未改动）
1. `CRAWL_FORBID_PROXY=true`，不增加任何代理/翻墙逻辑
2. `FORBIDDEN_HOSTS` 完整保留蒙古 `.gov.mn`；程序**不会**对 gov.mn 发起直接 HTTP
3. 允许 Google 查询字符串携带 `site:police.gov.mn` 等（仅搜索引擎快照）；结果若落在 gov.mn 仍被 `is_forbidden_url` 拦截
4. Web 面板、API、定时任务、邮件推送保持不变

### 模块变更摘要
| 模块 | 优化点 |
|------|--------|
| `config/drug_lexicon.py` | 中/蒙/英/俄长尾词与口岸组合检索扩充 |
| `config/core_official.py` | 媒体多语种任务增加；新增 gov.mn **快照检索** |
| `config/forum_search.py` | Reddit/知乎/贴吧等任务翻倍，本地话题补盲 |
| `config/global_media.py` | 路透/新华/CGTN/俄媒 + 中蒙边境专项 |
| `config/official_stats.py` | 海关/检察院多语种统计检索；PDF 关键词扩充 |
| `app/crawler/filters.py` | 正文强词可入库；无日期不丢；弱化弱词限制 |
| `app/crawler/search_feeds.py` | 站内搜索自动翻页（最多 5 页） |
| `app/crawler/engine.py` | 取消蒙通社标题强制；UA 池 + 抖动 |
| `app/crawler/rate_limit.py` | 蒙古本土媒体日限额放宽 |
| `app/analysis/engine.py` | 分类/30 日趋势/报告分模块展示适配增量 |

### 建议环境变量
见根目录 `.env.example`：`NEWS_WHEN=1y`、`CRAWL_MAX_DEPTH=2`、`CRAWL_MAX_PER_HOST_PER_DAY=10`、`ENABLE_FORUM_SEARCH=true`。

### 验证
```bash
cd mongolia-antidrug-intel
python -m pytest tests/test_core.py -q
```
通过后在面板触发「全量采集」或调用采集 API，观察情报条目是否明显增长。
