# 官方近30日情报归集（文档对齐说明）

## 为什么其他平台能拿到「官方最近消息」

| 做法 | 本系统此前 | 其他平台常见做法 |
|------|------------|------------------|
| 采集方式 | 主要靠 Google News 泛搜「Mongolia + drug」 | **直采** police/customs/montsame/gov 官网栏目 |
| 语言 | 偏英语聚合 | 蒙语西里尔关键词（хар тамхи 等） |
| 官网巡检 | `ENABLE_OFFICIAL_CRAWL=false` | 固定种子递归新闻/公告栏 |
| 结果 | 近30日英媒几乎 0 条可入库 | 能看到官网通报（哪怕条数仍少） |

蒙古国官方涉毒公开通报本身偏少；但**不直采官网**会把仅有的蒙语通报也漏掉。

## 已按文档落地的规则

1. **核心种子**（`config/core_official.py`）  
   Montsame、police.gov.mn、customs.gov.mn、UNODC Mongolia、gov.mn/mohs、zasag.mn  
2. **官网直采开启**：`ENABLE_OFFICIAL_CRAWL=true`；新闻监测轮也跑核心官网增量  
3. **时效**：`NEWS_WHEN/FULL_WHEN=30d`，`CRAWL_MAX_AGE_DAYS=30`  
4. **蒙语关键词补强**：тэмцэх、газар хил、сэргээх төв 等  
5. **报告格式**：七大模块归集 + 情报说明 + 综合交叉研判（近30日）  
6. **过滤**：剔除体育兴奋剂等噪声  

## 重要说明

文档中列举的具体案情（如「海岸运河」行动吨数、Playtime 查获细节等）**不会被系统伪造入库**。  
系统只采集各官网/授权公开渠道**真实存在**的页面；若官网未公开对应通报，报告对应模块会显示「本周期无新增」。
