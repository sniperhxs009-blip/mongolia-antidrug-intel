## 境外代理（解决国内 IP SSL 超时）
在 Render / `.env` 配置其一：

```env
CRAWL_PROXY_URL=http://user:pass@host:port
# 或多代理轮换
CRAWL_PROXY_URLS=http://p1:port,http://p2:port
```

未配置时代理为空（直连）；蒙古 gov 站在中国大陆可能仍超时。

## 限频
`CRAWL_MAX_PER_HOST_PER_DAY=3`：单站每日最多抓 3 次。
