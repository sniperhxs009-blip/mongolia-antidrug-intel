# 邮箱推送配置教程

## 功能

1. **每日简报**：定时生成研判报告并发送 HTML 邮件  
2. **即时告警**：检出专项行动/跨境缉毒/新法/口岸严查等信号后秒级推送  

## SMTP 配置（以 QQ 邮箱为例）

1. 登录 QQ 邮箱 → 设置 → 账户 → 开启 SMTP  
2. 生成授权码（不是登录密码）  
3. 写入 `.env`：

```env
SMTP_HOST=smtp.qq.com
SMTP_PORT=465
SMTP_USE_SSL=true
SMTP_USER=你的QQ号@qq.com
SMTP_PASSWORD=授权码
EMAIL_FROM=你的QQ号@qq.com
EMAIL_TO=接收邮箱@example.com
EMAIL_DAILY_BRIEF_HOUR=8
EMAIL_DAILY_BRIEF_MINUTE=0
ENABLE_ALERT_EMAIL=true
```

## 其他常见邮箱

| 服务商 | HOST | PORT | SSL |
|--------|------|------|-----|
| QQ | smtp.qq.com | 465 | true |
| 163 | smtp.163.com | 465 | true |
| Gmail | smtp.gmail.com | 465 | true（需应用专用密码） |
| Outlook | smtp.office365.com | 587 | false（STARTTLS） |

若使用 587 + STARTTLS：

```env
SMTP_PORT=587
SMTP_USE_SSL=false
```

## Render 上配置

在 Web Service → Environment 添加上述变量（`SMTP_PASSWORD` 等选 Secret）。

## 验证

- 控制台显示「邮箱：已配置」
- 触发巡检后查看 `email_logs`（API/数据库）或收件箱
- 未配置 SMTP 时系统仍正常采集与出报告，仅跳过发信

## 邮件内容结构

- 分机构归类
- 情报等级、类别、发布时间
- 来源网址可点击
- 紧急告警红色左边框突出显示
