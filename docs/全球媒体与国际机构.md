# 全球主流媒体与国际禁毒机构采集

体系 **system_id=10**：专门捕捉境外对蒙古国毒情的报道与机构通稿。

## 国际禁毒组织 / 执法机构

UNODC、INCB、WHO、WCO、INTERPOL、EUDA/EMCDDA、美国国务院 INL、DEA、Global Initiative / OC-Index

## 全球主流媒体（定向 site 搜索）

路透、美联社、法新/France24、BBC、卫报、CNN、半岛、外交官、自由亚洲、VOA、日经亚洲、南华早报、新华社、CGTN、塔斯社、俄新社、日本时报、韩联社、AKIpress、VICE、彭博、纽约时报、DW、Eurasianet 等

## 运行方式

- **每小时新闻监测**：精简国际机构 + 核心全球媒体 + 中英综合检索（近 7 天）
- **每日全量**：完整媒体/机构列表 + 俄语国际报道

所有查询强制 `Mongolia` + 毒品强词，并经严格涉毒过滤，避免无关国际新闻。
