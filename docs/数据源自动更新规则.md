# 数据源自动更新规则

## 覆盖范围（七大体系，缺一不可）

1. 国家级禁毒统筹委员会体系  
2. 刑事缉毒执法体系  
3. 麻精药品与制毒原料行业监管体系  
4. 边境海关缉毒查验体系  
5. 全国地方层级禁毒机构  
6. 戒毒康复医疗机构体系  
7. 国际禁毒协作机构（含 UNODC 蒙古公开信息）

种子清单见 `config/sources.py`。

## 自动发现规则

1. 仅允许白名单域名（`.gov.mn` 相关官方域、乌兰巴托门户、UNODC 等）  
2. 同主机/子域递归，最大深度 `CRAWL_MAX_DEPTH`（默认 3）  
3. 优先收录含 news/info/press/announcement/legal/prevention/cooperation 等栏目链接  
4. 新发现 URL 写入 `sources` 表（`is_seed=false`），后续巡检自动纳入  
5. 不可达源标记 `last_status=unreachable`，不删除以便回溯  

## 内容过滤

- 必须命中禁毒相关关键词（蒙/英/俄/中）才入库  
- 过滤无关政务/民生新闻  
- 过期旧闻（默认超过约 540 天）丢弃  
- URL + 内容哈希去重；内容变化标记 `updated`  

## 人工扩展

如官方域名变更，只需在 `config/sources.py` 增加/修正 `base_url` 与 `seed_paths`，重启后自动种子化并继续递归发现。

## 绝对禁止

- 自媒体、论坛、新闻搬运站、非官方资讯  
- 中国及其他第三国非授权站点（中蒙双边仅采信蒙方公开渠道与 UNODC）
