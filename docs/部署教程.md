# 部署教程

## 一、本地部署

### 1. 环境要求
- Python 3.11+
- 可访问蒙古国官方网站与 UNODC（如需翻译，需可访问翻译服务）

### 2. 安装

```bash
cd mongolia-antidrug-intel
python -m venv .venv

# Windows
.venv\Scripts\activate
# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt
copy .env.example .env   # Windows
# cp .env.example .env   # Linux/macOS
```

编辑 `.env`，至少配置邮箱相关变量（可选，不配则跳过发信）。

### 3. 启动

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

浏览器打开：http://127.0.0.1:8000

### 4. 单次跑批（不发邮件）

```bash
python scripts/run_once.py
```

### 5. Docker

```bash
docker build -t mongolia-antidrug-intel .
docker run -p 8000:8000 --env-file .env -v intel-data:/var/data mongolia-antidrug-intel
```

## 二、Render 部署

1. 将本仓库推送到 GitHub
2. 登录 [Render](https://render.com) → New → Blueprint
3. 选择本仓库，Render 会读取 `render.yaml` 自动创建 Web Service + 磁盘
4. 在 Environment 中填入 SMTP 相关密钥：
   - `SMTP_HOST` / `SMTP_PORT` / `SMTP_USER` / `SMTP_PASSWORD`
   - `EMAIL_FROM` / `EMAIL_TO`
5. 部署完成后访问 `https://<service-name>.onrender.com`

> Free 实例会休眠；唤醒后调度器会重新启动并尽快执行一轮巡检。建议将 `CRAWL_INTERVAL` 设为 `every_12h` 或 `daily`。

## 三、数据持久化

- 本地默认：`./data/intel.db`
- Render：`/var/data/intel.db`（已挂载磁盘）
- 报告文件：`data/reports/` 或 `/var/data/reports/`（可通过修改工作目录/环境保证可写）
