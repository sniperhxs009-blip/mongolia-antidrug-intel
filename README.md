# 蒙古国禁毒全网情报自动采集、定时抓取、邮箱推送、交叉研判分析系统

专属蒙古国禁毒领域全自动情报搜集研判系统。数据源**严格限定**蒙古国官方禁毒、缉毒、监管、边境、康复、国际协作机构官网及下级子站（含 UNODC 蒙古公开信息），不爬取自媒体与第三国非授权站点。

## 功能总览

| 模块 | 能力 |
|------|------|
| 全网采集引擎 | 增量爬虫、关键词过滤、蒙/英/俄→中翻译、去重、过期过滤 |
| 定时任务 | 每小时 / 6小时 / 12小时 / 每日；断点续爬；低频率防封 |
| 邮箱推送 | 每日简报 + 重大突发即时告警（HTML 结构化） |
| 交叉研判 | 七维研判 + 周/月趋势 + 风险预警 + 线索清单 |
| 报告输出 | 公安情报文风固定十章结构，Markdown/HTML 双格式 |
| Web 控制台 | 统计、触发巡检、改间隔、查数据源与报告 |

## 快速开始

```bash
python -m venv .venv
.venv\Scripts\activate          # Windows
pip install -r requirements.txt
copy .env.example .env
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

打开 http://127.0.0.1:8000

## 七大体系

1. 国家级禁毒统筹委员会体系  
2. 刑事缉毒执法体系  
3. 麻精药品与制毒原料行业监管体系  
4. 边境海关缉毒查验体系  
5. 全国地方层级禁毒机构  
6. 戒毒康复医疗机构体系  
7. 国际禁毒协作机构  

种子与白名单：`config/sources.py`

## API 摘要

- `GET /api/health` — 健康检查  
- `GET /api/intel` — 情报列表  
- `GET /api/sources` — 数据源  
- `GET /api/reports` — 报告列表  
- `POST /api/crawl/run?report_type=daily` — 触发采集研判
